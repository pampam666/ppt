# Multi-Agent Presentation Slide Generator

Welcome to the **Multi-Agent Presentation Slide Generator**! This system utilizes a multi-agent architecture to fully automate the process of creating presentation slides. From parsing raw PDF documents and analyzing Product Requirements Documents (PRDs) to conducting visual asset searches and generating final HTML slide decks, this tool handles it all seamlessly.

## 🛠️ Skills and Tools Used

This project relies on several powerful AI skills:
- **`power-design`**: The core engine for applying codified design principles and brand DNA to generate beautiful HTML slides.
- **`filesystem` (Skillfish)**: Used for reading and writing files within the workspace, ensuring the directory structure is maintained.
- **PDF to Markdown Extraction (Composio)**: Extracts text and tables from uploaded PDF reference documents and converts them into structured Markdown for agent analysis.
- **Web Search Images (Tavily)**: Context-aware internet search used to find high-quality, relevant supporting visuals for the presentation.

## 📂 Main Directory Structure

The system enforces a strict, organized workspace for every presentation project. All work is centralized under:
`<workspace>/slide/<presentation-title>/`

Inside each presentation's folder, you will find the following structure:
```text
slide/<presentation-title>/
├── docs/
│   ├── prd/               ← Place your Product Requirements Document (Goals & Target Audience) here.
│   └── product/           ← Autogenerated. Product details extracted from PDFs go here.
├── src/
│   ├── attachment/        ← Place user-uploaded raw files (reference PDFs, specific images) here.
│   └── structure/         ← Autogenerated. 3 drafted structure options will be saved here.
└── output/                ← Autogenerated. The final slides.html deliverable will be placed here.
```

## 🚀 Getting Started

To kick off the slide generation process for your project, simply use the following command in the chat:

```text
/slide-preparation-flow
```

The system will guide you through data extraction, PRD analysis, structure selection, and final slide rendering. 

For detailed, step-by-step instructions on how to prepare your workspace before running this command, please refer to the [GUIDE.md](GUIDE.md).
